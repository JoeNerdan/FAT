Usage

Basic Rules

The tool offers basic rules. They are maintained by the FITML-Team, can not be deleted and mostly check your FITML for errors that will break the processing mechanism.

User Rules

You are able to create your own rules.

Add Rules

A JS-Api is used to define new rules. The syntax is: FAT.addTest(Testname, xpathToBeExecuted, callbackFunction, message ,type);

Delete Rules

To delete a rule, simply click the delete button next to it.

Testrun

The FITML code to be checked is pasted in the textarea.
